Introduction

CSS stands for Cascading Style Sheets and is a language used to modify the look and layout of a webpage. It has many uses, from changing the size,color, kerning, and spacing of text to creating an entire background from code. It has a very simple syntax.

CSS can be broken down into 6 basic categories: Pseudo-classes, properties, functions, data-types, concepts, and @rules. This webpage will focus only on some of the basic Properties.

Properties control the color,size,direction, and other visual style options of text, images, videos, and many other website elements.

background

The background property is a shorthand property for setting most background properties at the same place in the style sheet. Just like other background properties, it can take multiple comma-seperated values, where each value ccorresponds to a background layer. The values it can control are background-image, background-position, background-size, background-repeat, background-origin, background-clip, background-attachment, and background-color.

The following is the default values for each background property:

  • background-image: none
  • background-position: 0% 0%
  • background-size: auto auto
  • background repeat: repeat
  • background-origin: padding-box
  • background-clip: border-box
  • background-attachment: scroll
  • background-color: transparent

An example of using the background property may look like this:

.element { background: rgba(255, 255, 255, 0) url(path/to/image.jpg) bottom 50% / contain fixed repeat; }
background-color

The background-color property is used to set the background color of an element. It sets a background color using a < color > value. If you set the background-image property of an element it will be layed over the background-color, however if the background is transparent or translucent, you will be able to see the background-color through the background image.

An example of using the background-color property may look like this:

.element { background-color: orange; }
background-image

The background-image property is used to set the background image of an element. Surprise! It can accept either one image value, or muliple comma-seperated values. If there is more than one background set, then each of the images is sized, positioned, and tiled according to the corresponding value in the other background properties. The images are then stacked on top of each other, with the first being on top of the stack.

An example of using the background-image property may look like this:

.element { background-image: url(path/to/img.jpg); }
border

Border is another shorthand property. It controls the width, style, and color of the top,right, bottom, and left borders of a box. The border property itself can only set the top,right, bottom, and left of the border to the same values. If you would like to set them to different values you may use other border properties.
